options:
参数名 类型 默认值 说明
auto boolean true 是否自动开始变色
type string text 目标变色类型：text、background、border
changeChildren boolean false 是否以所有子元素作为变色对象，true且innerHTML为纯文字时，单个字符将被添加span标签
randomcolor boolean false 是否随机生成颜色
normalcolor string #000 pause恢复时目标的颜色
colors array [ '#00ffff', '#f1c40f', '#3498db', '#000000', '#1abc9c', '#e67e22', '#95a5a6', '#ffffff', '#2ecc71', '#f00000' ] 指定颜色集合，randomcolor=false时有效
randomshow boolean false 是否随机展示colors指定的颜色集合
pause boolean true 是否支持点击目标暂停/恢复
